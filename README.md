# Badminton_Tournament_Engine
Object-oriented C++ tournament management system with singles &amp; doubles support as well as different player categories like Beginners, Intermediate and Advanced. It will also track the match history and player statistics.
A console-based badminton tournament management system built using C++.  
Supports singles and doubles matches with a dynamically maintained ranking system implemented using a custom doubly linked list.

---

## Features

- Player registration (Singles & Doubles)
- Doubles partner linking system
- Match simulation engine
- Automatic points calculation
- Player Statistics
- Store and display matches history
- Dynamically sorted ranking table
- Menu-driven CLI interface
- Modular header/source architecture

---

## Concepts Demonstrated

- Object-Oriented Programming (OOP)
- Custom Singly Linked List Implemenation
- Custom Doubly Linked List Implementation
- Dynamic Memory Management
- Pointer Safety Handling
- Ordered Insertion Algorithm
- Tournament System Design
- Live match simulation
